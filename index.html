What is Web Development?

It's about creating and developing websites for hosting via Internet. Once you have created the website it requires a place to stay,
let's say home, now this home is technically called a Web Server which is a computer or system that stores,
processes and delivers website files to Web Browsers (Chrome,Safari,Edge,Firefox etc). This entire process is called Web Hosting.

Front-end Web Development

Front-end Web Development is the development of the graphical user interface of a website, through the use of HTML, CSS, and Javascript, so that users can interact with that website. In simple terms, its the outer layer of a website which the users can see.

Back-end Web Development

Back-end development covers server-side web application logic and integration and activities, like writing APIs, creating libraries, and working with system.
Backend developers take care and maintain the back-end of a website, including databases, servers, and apps, and they control what you don't see.
